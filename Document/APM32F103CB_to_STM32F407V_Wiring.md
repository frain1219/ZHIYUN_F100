# APM32F103CBT6 <-> STM32F407VGT6 接线规范（逆向侧车）

> 版本：2026-05-01（LQFP100引脚重映射版）  
> 目标：F103 运行量产固件，F407 作为高阻监听/可控注入侧车，不替代 F103 主控职责。  
> 场景：逆向取证、阻塞解除实验、行为一致性对照。

---

## 0. 重要说明：STM32F407VGT6 (LQFP100) 封装引脚限制

**STM32F407VGT6 采用 LQFP100 封装，存在以下引脚限制：**

### 不可用引脚

- **GPIOG**: PG0-PG15 在 LQFP100 封装中**全部未引出**
- **GPIOF**: PF0-PF15 在 LQFP100 封装中**全部未引出**

### 引脚重映射方案

为适应 LQFP100 封装限制，本文档已对原 PG 和 PF 引脚进行重新映射：

| 原引脚 | 新引脚 | 功能模块 | 说明 |
|--------|--------|---------|------|
| PG2 | PC0 | 按键输入模块 | F103 PB4 右键输入 |
| PG5 | PC1 | 编码器模块 | F103 PB5 MO_PWM |
| PG6 | PC2 | 编码器模块 | F103 PB6 编码器A相 |
| PG7 | PC3 | 编码器模块 | F103 PB7 编码器B相 |
| PF7 | PD2 | 副板控制模块 | F103 PB12 AUX |
| PF8 | PD3 | 副板控制模块 | F103 PB13 AUX/STAT |
| PF9 | PD4 | 副板控制模块 | F103 PB14 EN测试点 |
| PF10 | PD5 | 副板控制模块 | F103 PB15 ADIM测试点 |
| PF11 | PD6 | 副板控制模块 | F103 PA8 W_PWM |
| PF12 | PD7 | 副板控制模块 | F103 PA9 C_PWM |
| PF14 | PC4 | 调试口监听模块 | F103 PA13 SWDIO |
| PF15 | PC5 | 调试口监听模块 | F103 PA14 SWCLK |

**注意：** 如果您使用的是 LQFP144 或更大封装的 STM32F407，可以恢复使用原 PG/PF 引脚定义，但需要同步修改固件中的引脚配置。

## 1. 强制安全规则

1. 两板必须共地（GND 至少 1 根，建议多根）。
2. 修改杜邦线必须断电后进行，禁止热插拔。
3. F407 默认配置为高阻输入；仅白名单引脚允许短时注入。
4. 禁止把 F407 推挽输出长期对驱 F103 输出。
5. 数字线默认串联 100 欧（每条线只串一次）。
6. 模拟采样线禁止串联 100 欧（避免 ADC 读值失真）。
7. 禁止用杜邦线直连以下信号到 F407 普通 GPIO：
   - `NRST`
   - `PA11/PA12`（USB D-/D+）
   - `PD0/PD1`（HSE）
   - `PC14/PC15`（LSE）

---

## 2. 连接定位

- 以 **芯片引脚网络名** 为准，不以排针丝印猜测为准。
- 接线前后都要做三件事：
  - 验证 GND 连通
  - 逐条核对 `F103信号 -> F407信号`
  - 上电后先空载观察（发热、复位异常、电流异常）

---

## 3. 按功能模块接线（便于排查）

> 这些信号在 `ZHIYUN_F100_Firmware/stm32f407vgt6_probe/src/probe_pins.c` 已加入监听表（`LIST_PINS` 可见）。  
> 建议按模块分批接线、分批验证：每完成一个模块就先上电观察，再继续下一个模块。

### 3.1 系统状态与控制链路（基础优先）

| F103 信号 | 功能语义 | F407 侧 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PC13` | PD 状态机关键输入 | `PE0` | 数字监听 | 是 |
| `PA0` | 供电路径相关控制 | `PE5` | 数字监听/注入 | 是 |
| `PB11` | PD GPIO 控制 | `PE1` | 数字监听/注入 | 是 |
| `PB3` | VS_EN 相关 | `PE2` | 数字监听/注入 | 是 |

### 3.2 按键输入模块

| F103 信号 | 功能语义 | F407 侧 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PA1` | 按键采样路径 | `PE6` | 数字监听/注入 | 是 |
| `PB4` | 右键输入 | `PC0` | 数字监听/注入 | 是 |

### 3.3 OLED 抓取模块（只监听）

#### 必选线（建议先接）

| F103 信号 | 功能语义 | F407 侧建议 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PA5` | OLED SPI SCK 候选 | `PE8` | 数字监听（建议 EXTI） | 是 |
| `PA7` | OLED SPI MOSI 候选 | `PE9` | 数字监听（建议 EXTI） | 是 |
| `PB10` | OLED CS 低有效候选 | `PE10` | 数字监听（建议 EXTI） | 是 |

#### 可选增强线（二阶段接入）

| F103 信号 | 功能语义 | F407 侧建议 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PB2` | OLED RES 候选 | `PE11` | 数字监听 | 是 |
| `PA6` | OLED D/C 或 select-line 候选（待 LA 验证） | `PE12` | 数字监听 | 是 |

OLED 模块注意事项：

1. OLED 抓取全部为监听，不允许在这些线上启用注入输出。
2. 先完成 `PA5/PA7/PB10` 三线，确认可形成事务边界后再加 `PB2/PA6`。
3. `PB6` 与“编码器模块”存在复用关系（量产固件亦可能在上电阶段将其作为 SPI 辅助选择线）；本规范的侧车接线中 **不再**将 `PB6` 用作 OLED 抓取线，避免与编码器 A 相监听冲突。
4. USB 供电与 OLED 线束接好后，先跑 30 秒纯监听，不执行任何注入命令，确认无异常再开始抓取。

### 3.4 编码器与 PWM 模块

| F103 信号 | 功能语义 | F407 侧 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PB5` | MO_PWM（下） | `PC1` | 数字监听（建议 EXTI） | 是 |
| `PB6` | 编码器 A 相 | `PC2` | 数字监听（建议 EXTI） | 是 |
| `PB7` | 编码器 B 相 | `PC3` | 数字监听（建议 EXTI） | 是 |
| `PA2` | RGB PWM 相关输出 | `PE7` | 数字监听 | 是 |

编码器/PWM 模块注意事项：

- `PB5/PB6/PB7` 若只靠 1ms 轮询可能漏掉窄脉冲；当前固件已为 `PC1/PC2/PC3` 打开 EXTI（边沿捕获），建议接线后用 `SUBSCRIBE ON` + `GET_STATS` 观察 `event_exti_count` 是否增长。
- 若本轮以 OLED 抓取为主且需要占用 `PB6`，请暂停接入编码器 A 相，避免冲突。

### 3.5 模拟采样模块（ADC）

| F103 信号 | 功能语义 | F407 侧 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PA3` | ADC 输入 | `PA0` | 模拟采样 | 是 |
| `PA4` | ADC 输入 | `PA1` | 模拟采样 | 是 |
| `PB0` | ADC 输入 | `PA2` | 模拟采样 | 是 |
| `PB1` | ADC 输入（VBAT） | `PA3` | 模拟采样 | 是 |

**注意：** 由于LQFP100封装限制，原PF3-PF6引脚未引出，已重映射到PA0-PA3。ADC功能已从ADC3迁移到ADC1。

### 3.6 副板控制与辅助分支模块

| F103 信号 | 功能语义 | F407 侧 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PB12` | AUX（IDR 读/辅助分支） | `PD2` | 数字监听/注入 | 是 |
| `PB13` | AUX/STAT（辅助分支） | `PD3` | 数字监听/注入 | 是 |
| `PB14` | EN 测试点（副板） | `PD4` | 数字监听/注入（含 emu/注入） | 是 |
| `PB15` | ADIM 测试点（副板） | `PD5` | 数字监听/注入（含 emu/注入） | 是 |
| `PA8` | W_PWM（副板） | `PD6` | 数字监听/注入（含 emu/注入） | 是 |
| `PA9` | C_PWM（副板） | `PD7` | 数字监听/注入（含 emu/注入） | 是 |

副板模块注意事项：

- `PD2..PD7` 既可监听也可注入，接线前请确认不会与 F103 的主动输出长期冲突（遵守“短时注入、可释放回高阻”的原则）。

### 3.7 调试口监听模块（只监听）

| F103 信号 | 功能语义 | F407 侧 | 类型 | 100 欧 |
|---|---|---|---|---|
| `PA13` | SWDIO | `PC4` | 仅监听，禁止驱动 | 是 |
| `PA14` | SWCLK | `PC5` | 仅监听，禁止驱动 | 是 |

## 4. 不建议接入项

- 电源相关：`VBAT`、`VDD`、`VDDA`、`VSS/VSSA`（仅共地，不做 GPIO 连接）
- 晶振与 USB 专用脚：见第 1 节
- 任何你还未在文档中确认语义的脚：先只监听，后再考虑注入
