## 1. 核心特性概览

### 1.1 内核与性能

| 特性 | 参数 |
|------|------|


### 4.2 时钟配置要点

- **系统时钟源选择**: HSICLK(8MHz)、HSECLK(4~16MHz)、PLLCLK
- **USBD时钟**: 必须为48MHz，系统时钟只能是48MHz、72MHz、96MHz之一
- **ADC时钟**: 最大14MHz，通过分频器配置
- **CSS(时钟安全系统)**: 检测HSECLK失效，自动切换到HSICLK

### 4.3 低功耗模式时钟行为

| 模式 | CPU时钟 | 外设时钟 | 唤醒源 |
|------|---------|----------|--------|
| 睡眠 | 关闭 | 运行 | 任意中断 |
| 停机 | 关闭 | 关闭 | EINT、PVD、RTC闹钟、USBD唤醒 |
| 待机 | 关闭 | 关闭 | NRST、IWDT、WKUP引脚、RTC闹钟 |

## 5. 启动配置

### 5.1 启动模式选择

启动时通过BOOT引脚选择启动模式：

| BOOT1 | BOOT0 | 启动模式 | 地址映射 | 应用场景 |
|-------|-------|----------|----------|----------|
| X | 0 | 主Flash | 0x0800 0000 | 正常运行 |
| 0 | 1 | 系统存储器 | 0x1FFF F000 | ISP编程(USART1) |
| 1 | 1 | 内部SRAM | 0x2000 0000 | 调试/临时运行 |

### 5.2 Bootloader

- 存放于系统存储器(0x1FFF F000)
- 支持通过USART1对Flash重新编程
- 可用于固件更新和恢复

## 6. 外设功能描述

### 6.1 NVIC(嵌套向量中断控制器)

| 特性 | 参数 |
|------|------|
| 可屏蔽中断通道 | 47个(不含16个Cortex-M3中断线) |
| 优先级位数 | 16级 |
| 中断延迟 | 低延迟向量传递 |

### 6.2 EINT(外部中断/事件控制器)

- 19个边沿检测器
- 触发方式：上升沿、下降沿、双边沿
- 最多80个GPIO可映射到16个外部中断线

### 6.3 DMA

- 7个独立可配置通道
- 支持存储器到存储器、外设到存储器、存储器到外设
- 支持的外设：SPI、I2C、USART、TMRx、ADC

### 6.4 定时器

#### 6.4.1 定时器功能比较

| 特性 | SysTick | TMR2/3/4 | TMR1 |
|------|---------|----------|------|
| 计数器分辨率 | 24位 | 16位 | 16位 |
| 计数器类型 | 向下 | 向上/向下/双向 | 向上/向下/双向 |
| 预分频系数 | - | 1~65536 | 1~65536 |
| 捕获/比较通道 | - | 4 | 4 |
| 互补输出 | - | 无 | 有(3对) |
| DMA请求 | - | 支持 | 支持 |

#### 6.4.2 高级定时器TMR1特性

- 死区插入
- 紧急刹车功能
- 互补PWM输出(0~100%调制能力)

### 6.5 看门狗

| 特性 | IWDT(独立看门狗) | WWDT(窗口看门狗) |
|------|------------------|------------------|
| 计数器分辨率 | 12位 | 7位 |
| 计数器类型 | 向下 | 向下 |
| 时钟源 | LSICLK(40KHz) | 主时钟 |
| 预分频系数 | 1~256 | - |
| 工作模式 | 停机/待机模式下可用 | 主时钟驱动 |
| 早期预警 | 无 | 有(中断) |

### 6.6 通信接口

#### 6.6.1 USART

| 接口 | 最大速率 | 特性 |
|------|----------|------|
| USART1 | 4.5Mbit/s | CTS/RTS、ISO7816、IrDA、LIN |
| USART2/3 | 2.25Mbit/s | CTS/RTS、ISO7816、IrDA、LIN |

#### 6.6.2 SPI

- 最大传输速率：18Mbps
- 支持主/从模式
- 每帧8位或16位
- 支持DMA

#### 6.6.3 I2C

- 支持标准模式(100KHz)和快速模式(400KHz)
- 支持7位和10位寻址
- 硬件CRC发生器/校验器
- 支持SMBus/PMBus

#### 6.6.4 CAN

- 兼容CAN 2.0A和2.0B(主动)
- 最大速率：1Mbit/s
- 3个发送邮箱
- 2个接收FIFO
- 14个可调节滤波器

#### 6.6.5 USBD

- USB 2.0 Full Speed Device(12Mbps)
- USBD1和USBD2共用寄存器和引脚
- USBD与CAN可同时工作(需特殊配置)

**USBD与CAN同时使用配置**:
1. 在USBD基地址偏移0x100处写0x00000001
2. PA11/PA12给USBD用，CAN复用其他引脚

#### 6.6.6 QSPI

- 支持单线、双线、四线SPI模式
- 8 bytes发送FIFO
- 8 bytes接收FIFO
- 支持8位、16位、32位访问

### 6.7 ADC

- 2个独立的12位ADC
- 最多16个外部通道
- 单次模式和扫描模式
- 支持DMA
- 双采样和保持功能
- 模拟看门狗功能
- 内部温度传感器(ADC1_IN16)

**ADC特性**:
| 参数 | 值 |
|------|-----|
| ADC时钟频率 | 0.6~14MHz |
| 采样速率 | 0.05~1MHz |
| 转换电压范围 | 0~VREF+ |
| 采样时间 | 1.5~239.5周期 |
| 总转换时间 | 14~252周期(含采样) |

### 6.8 CRC计算单元

- 计算32位数据字的CRC码
- 用于软件签名验证

## 8. 调试接口

### 8.1 SWJ-DP调试接口

支持两种调试方式：

| 接口 | 引脚数 | 引脚 | 说明 |
|------|--------|------|------|
| JTAG | 5 | TMS/SWDIO, TCK/SWCLK, TDI, TDO, NJTRST | 标准JTAG |
| SWD | 2 | SWDIO, SWCLK | 串行调试(推荐) |

### 8.2 调试引脚复用

| 引脚 | JTAG功能 | SWD功能 | GPIO复用 |
|------|----------|---------|----------|
| PA13 | TMS | SWDIO | - |
| PA14 | TCK | SWCLK | - |
| PA15 | TDI | - | TMR2_CH1_ETR/SPI1_NSS |
| PB3 | TDO | - | TRACESWO/TMR2_CH2/SPI1_SCK |
| PB4 | NJTRST | - | TMR3_CH1/SPI1_MISO |

### 8.3 调试模式行为

- 调试模式下定时器可被冻结
- PWM输出可被禁止
- 看门狗可被暂停

## 9. 中断系统

### 9.1 Cortex-M3内核中断

| 异常类型 | 位置 | 优先级 | 说明 |
|----------|------|--------|------|
| Reset | 1 | -3(最高) | 复位 |
| NMI | 2 | -2 | 不可屏蔽中断 |
| HardFault | 3 | -1 | 硬件错误 |
| MemManage | 4 | 可编程 | 内存管理错误 |
| BusFault | 5 | 可编程 | 总线错误 |
| UsageFault | 6 | 可编程 | 用法错误 |
| SVCall | 11 | 可编程 | 系统服务调用 |
| Debug Monitor | 12 | 可编程 | 调试监控 |
| PendSV | 14 | 可编程 | 挂起系统服务 |
| SysTick | 15 | 可编程 | 系统滴答定时器 |

## 附录：逆向分析要点

### A. 固件加载地址

- **Flash起始地址**: 0x0800 0000
- **Flash大小**: 128KB (0x20000)
- **SRAM起始地址**: 0x2000 0000
- **SRAM大小**: 20KB (0x5000)

### B. 中断向量表

- **默认位置**: 0x0800 0000 (Flash起始)
- **向量表偏移寄存器**: VTOR (可重定位)

### C. 常见固件特征

- **启动代码**: 通常位于0x0800 0000
- **中断向量表**: 前256字节
- **Bootloader**: 0x1FFF F000 (系统存储器)

### D. 调试接口访问

- **SWD**: 推荐使用，仅需2根线
- **JTAG**: 完整调试，5根线
- **调试端口基地址**: 0xE000 0000 (CoreSight)

### E. 安全特性

- **选项字节**: 0x1FFF F800
- **读保护**: 可通过选项字节配置
- **写保护**: 每页可独立配置
