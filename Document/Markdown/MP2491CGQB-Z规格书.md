# MP2491CGQB-Z 规格书 (固件逆向版)

> **文档说明**: 本文档仅保留对APM32F103CBT6固件逆向有用的技术信息,重点保留VSEL1/VSEL2电压选择、EN使能控制、PG状态监测等与MCU交互的引脚。

---

## 一、芯片概述

**型号**: MP2491C  
**类型**: 高压降压转换器 (Buck Converter)  
**封装**: QFN-13 (2.5mm×3mm)

### 核心特性
- 输入电压范围: 4.0V ~ 32V
- 输出电压范围: 0.5V ~ 30V
- 最大输出电流: 6A
- 开关频率: 490kHz (固定)

## 三、控制引脚详解 (逆向核心)

### 3.1 输出电压选择 (VSEL1/VSEL2)

**真值表**:

| VSEL1 | VSEL2 | 参考电压 (VREF) | 典型输出电压 | 应用场景 |
|-------|-------|----------------|-------------|----------|
| 0 (AGND) | 0 (AGND) | 2.0V | 20V | 高压输出 |
| VCC | 0 (AGND) | 1.5V | 15V | 中高压输出 |
| 0 (AGND) | VCC | 0.9V | 9V | 中压输出 |
| 1/2×VCC | 0 (AGND) | 1.2V | 12V | 标准输出 |
| VCC | VCC | 0.5V | 5V | USB标准电压 |

**逆向要点**:
- VSEL1支持三种状态: AGND、VCC、1/2×VCC
- VSEL2支持两种状态: AGND、VCC
- **引脚不能悬空**
- 状态切换有20μs去抖时间
- 可通过GPIO动态控制实现电压切换

### 3.2 使能控制 (EN)

| EN电压 | 状态 | 阈值 |
|--------|------|------|
| > 1.24V | 芯片使能 | 上升阈值 |
| < 0.96V | 芯片关闭 | 下降阈值 |

**逆向要点**:
- 内部有2MΩ下拉电阻到地
- 通过GPIO控制使能/关闭

### 3.3 工作模式选择 (MODE)

| MODE引脚电压 | 工作模式 | 特点 |
|-------------|---------|------|
| 0 ~ 0.12×VCC | 强制PWM模式 | 固定频率,适合重载 |
| 0.9×VCC ~ VCC | 自动PFM/PWM | 轻载降频,高效率 |

**逆向要点**:
- MODE引脚不能悬空
- 支持动态切换

### 3.4 电源状态指示 (PG)

**PG状态表**:

| 条件 | PG状态 | 说明 |
|------|--------|------|
| 输出电压正常 | 高阻 (需上拉) | 93%×VREF < VOUT < 113%×VREF |
| 输出欠压 | 低电平 | VOUT < 87%×VREF |
| 输出过压 | 低电平 | VOUT > 107%×VREF |
| EN=0 | 低电平 | 芯片关闭 |

**PG时序**:
- 上升延迟: 250μs (软启动完成后)
- 下降延迟: 50μs (检测到异常)
