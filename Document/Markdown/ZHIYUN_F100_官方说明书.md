# 智云五倍 F100 官方说明书 (固件逆向版)

> **文档说明**: 本文档仅保留对APM32F103CBT6固件逆向有用的技术信息,包括按键操作逻辑、工作模式、固件升级流程等。删除了产品外观、光照度参数等用户信息。

---

## 一、产品概述

**产品型号**: PL104  
**产品名称**: 智云五倍 F100 手持棒灯  
**核心特性**:
- 最大功率: 100W
- 色温范围: 2700K ~ 6200K
- RGB色域: 0° ~ 360°
- 显色指数: CRI ≥ 96, TLCI ≥ 97
- 灯珠数量: 176PCS 冷暖 + 148PCS RGB

## 三、工作模式详解 (逆向核心)

### 3.1 CCT 模式 (色温/亮度)

**进入方式**: 单击 DIM 模式键

**可调参数**:
- 色温: 2700K ~ 6200K
- 亮度: 0% ~ 100%


---

### 3.2 HSI 模式 (色相/饱和度/亮度)

**进入方式**: 单击 HSI 模式键

**可调参数**:
- 色相 (H): 0° ~ 360°
- 饱和度 (S): 0% ~ 100%
- 亮度 (I): 0% ~ 100%


---

### 3.3 FX 光效模式

**进入方式**: 单击 FX 光效模式键

**可调参数**:
- 光效类型
- 亮度: 0% ~ 100%



---

### 3.4 全功率模式

**进入方式**: 
1. 进入 CCT 模式
2. 单击 MAX 全功率键

**特性**:
- 最大功率输出 (100W)
- 可调节色温和亮度
- 温度过高时触发高温保护



---

## 四、固件升级系统 (逆向关键)

### 4.1 升级工具

**工具名称**: Zhiyun Gimbal Tools  
**所需文件**:
- USB 驱动 (Windows 系统)
- 智云校准升级工具
- 固件文件 (.ptz 格式)

### 4.2 升级流程

```
1. 下载文件
   ├─ USB 驱动 (Windows)
   ├─ Zhiyun Gimbal Tools
   └─ 固件文件 (.ptz)

2. 安装驱动
   └─ Windows 系统需要安装 USB 驱动
   └─ macOS 系统无需安装

3. 准备固件
   └─ 解压固件压缩包
   └─ 获取 .ptz 文件

4. 进入升级模式
   ├─ 关闭棒灯
   ├─ 按住 MAX 全功率键
   ├─ 长按电源键开机
   └─ 自动进入升级模式

5. 连接电脑
   └─ USB Type-C 数据线连接棒灯和电脑

6. 执行升级
   ├─ 打开 Zhiyun Gimbal Tools
   ├─ 选择对应端口
   ├─ 点击"打开串口"
   ├─ 点击"固件更新"
   ├─ 点击"浏览"选择 .ptz 文件
   └─ 点击"更新"

7. 升级完成
   └─ 棒灯自动关机
```

### 4.3 升级模式触发条件

**按键组合**: MAX 全功率键 + 电源键 (长按)  
**触发时机**: 关机状态下同时按住


### 4.4 固件文件格式

**扩展名**: .ptz

**推测内容**:
- 固件二进制数据
- 校验信息(CRC32或MD5)
- 版本信息
- 加密/压缩数据(可能)

## 六、安全保护机制

### 6.1 温度保护

```
正常工作温度: -10℃ ~ 40℃

温度过高 → 高温保护机制启动
  ├─ 风扇全速运转
  ├─ 降低输出功率
  └─ 必要时自动关机
```


### 6.2 电池保护

**充电温度范围**: 0℃ ~ 45℃

**保护机制**:
- 过充保护
- 过放保护
- 温度保护

## 八、逆向推理总结

### 8.1 固件关键模块

| 模块 | 功能 | 推测实现 |
|------|------|---------|
| 按键处理 | 检测按键事件 | GPIO中断 + 消抖 |
| 编码器处理 | 检测旋转方向和速度 | 定时器编码器模式 |
| 状态机 | 管理工作模式切换 | switch-case状态机 |
| 参数调节 | 处理参数变化 | 范围限制 + 步进调节 |
| PWM控制 | LED亮度/色温控制 | 定时器PWM输出 |
| I2C通信 | HUSB238通信 | I2C读写函数 |
| OLED显示 | 参数显示 | I2C/SPI通信 |
| 状态记忆 | 保存/恢复状态 | Flash/EEPROM读写 |
| 固件升级 | Bootloader | 串口通信 + Flash写入 |
| 温度保护 | 过热保护 | ADC读取 + 风扇控制 |
| 电池保护 | 过充/过放保护 | ADC读取 + 充电控制 |
